MDN Forms
-This lesson talked about forms and having clients fill it
- It is not important but most UI and UX designers use it

- "built in form validation" is when you can use attributes like: required, minlength, maxlength, etc
- use pseudo elements (input:invalid, input:invalid:required, etc)
- For number fields (i.e. type="number">)
- tHTMLButtonElement (represents a button element)
- Most browsers support the Constraint Validation API, which consists of a set of methods and properties available on many form element DOM interfaces
